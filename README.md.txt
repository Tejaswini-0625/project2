Dataset

The dataset used in this project is the BreakHis (Breast Cancer Histopathological Database).

Due to its large size, it is not included in this repository.

You can download it from:
https://www.kaggle.com/datasets/ambarish/breakhis

After downloading, place the dataset in the project directory like this:

project/
│── BreakHis_v1/
│── Main.py
